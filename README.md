# LinkedIn Revenue Engine

Turn LinkedIn into a client acquisition machine.

This plugin provides high-performance commands to:
- Generate outbound sequences
- Reply to DMs
- Convert comments into leads
- Book meetings
- Improve offer positioning

## Commands

- /linkedin-sequence
- /dm-reply
- /comment-reply
- /rdv-booking-script
- /offer-angle

## Installation

Import this plugin into Claude and use commands directly in your conversations.
